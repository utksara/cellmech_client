CellFlow

Description

Cellflow is a web application to synthezie and replicate cellular/microbiological simulation.
In a nutshell it aims to provide an interface to simulate mesospcopic phenonemena which can
be studied under classical mechanics and micro-fluid dynamics. The app is built on top of a
dataflow programming based library providing APIs to design interconnected systems

Features

  • Design micro-biological system from scratch in a planar medium
  • Design virtual experiment of micro-biology
  • Use predefined, commonly used equations to run your experiment/simulation or define your own
  • Create steady state and real time simulation of your system
  • (coming soon) Save your data, analyse it, create dashboard
  • (coming soon) Directly replicate your tangible experimental setup into simulation by uploading images of your experiment
