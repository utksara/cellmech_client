Description

Beehive is a web application to graph based simulation.
In a nutshell it aims to provide an interface to simulate any system
which can be modelled as a directed graph. The app utilizes
dataflow programming to design interconnected graphical systems

Features

  • Design micro-biological system from scratch in a planar medium
  • Design virtual experiment of micro-biology
  • Use predefined, commonly used equations to run your experiment/simulation or define your own
  • Create steady state simulation of your system
  • (coming soon) Create real time simulation of your system
  • (coming soon) Save your data, download it
  • (coming soon) Save your SYSTEM in beehive repository and access it from any beehive project
